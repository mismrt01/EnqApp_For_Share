Plan: Clone Protection & License Activation System for EnqBoss
Context
The developer shared this repo publicly on GitHub. The app (EnqBoss — React + Vite + Supabase B2B sales ops tool) has no ownership identity or access control for cloners. The goal is to implement a real offline license activation system where:

Any clone gets a 30-day free trial automatically
After the first user login, the app generates a unique install hash and emails it to imharsh080@gmail.com via the existing Gmail API
After 30 days, the app blocks access and shows a "enter your license key" screen
The admin (imharsh080@gmail.com) receives the hash, converts it to an alphanumeric activation code with a duration suffix, and sends it back to the user
The user enters the code; the app decodes and verifies it, then activates for the specified duration
Admin email (hardcoded owner): imharsh080@gmail.com
Trial period: 30 days from first login

Implementation Details
1. src/lib/license.ts — License engine (new file)
const ADMIN_EMAIL = 'imharsh080@gmail.com';
const TRIAL_DAYS = 30;
const INSTALL_ID_KEY = 'enqboss_install_id';      // stores the install hash
const INSTALL_TS_KEY = 'enqboss_install_ts';       // stores first-login timestamp (obfuscated)
const LICENSE_KEY = 'enqboss_license';             // stores activation data after key entry

export type LicenseStatus = 'ok' | 'trial' | 'expired' | 'licensed';

// Generate install hash: SHA-256(supabaseUrl + timestamp + userAgent slice)
export async function generateInstallHash(supabaseUrl: string): Promise<string> {
  const fingerprint = supabaseUrl + navigator.userAgent.slice(0, 20);
  const buf = await crypto.subtle.digest('SHA-256', new TextEncoder().encode(fingerprint));
  return Array.from(new Uint8Array(buf)).map(b => b.toString(16).padStart(2,'0')).join('');
}

// Called on first login — idempotent
export async function initLicense(supabaseUrl: string): Promise<string> {
  let hash = localStorage.getItem(INSTALL_ID_KEY);
  if (!hash) {
    hash = await generateInstallHash(supabaseUrl);
    const tsObf = btoa(String(Date.now() ^ 0xDEADBEEF));
    localStorage.setItem(INSTALL_ID_KEY, hash);
    localStorage.setItem(INSTALL_TS_KEY, tsObf);
  }
  return hash;
}

// Read install timestamp
function getInstallTs(): number {
  const raw = localStorage.getItem(INSTALL_TS_KEY);
  if (!raw) return Date.now();
  try { return parseInt(atob(raw), 10) ^ 0xDEADBEEF; } catch { return Date.now(); }
}

export function getDaysRemaining(): number {
  const elapsed = (Date.now() - getInstallTs()) / 86400000;
  return Math.max(0, Math.ceil(TRIAL_DAYS - elapsed));
}

// Check license status (admin always 'ok')
export function getLicenseStatus(userEmail?: string | null): LicenseStatus {
  if (userEmail === ADMIN_EMAIL) return 'ok';
  
  // Check if a valid activation key has been entered
  const act = localStorage.getItem(LICENSE_KEY);
  if (act) {
    try {
      const { activatedAt, durationDays } = JSON.parse(atob(act));
      if (durationDays === 0) return 'licensed'; // lifetime
      const elapsed = (Date.now() - activatedAt) / 86400000;
      if (elapsed < durationDays) return 'licensed';
    } catch {}
  }

  // Fall back to trial
  const daysLeft = getDaysRemaining();
  return daysLeft > 0 ? 'trial' : 'expired';
}

// Activation key format: <8-char-base36-hash-prefix>-<3-digit-duration>
// e.g. "K7X2M9PQ-030"
export function activateLicense(key: string): { ok: boolean; error?: string } {
  const installHash = localStorage.getItem(INSTALL_ID_KEY);
  if (!installHash) return { ok: false, error: 'No install record found.' };

  const match = key.trim().toUpperCase().match(/^([A-Z0-9]{8})-(\d{3})$/);
  if (!match) return { ok: false, error: 'Invalid key format. Expected XXXXXXXX-DDD.' };

  const [, keyPrefix, durationStr] = match;
  
  // Encode our hash prefix to base36 for comparison
  const hashNum = parseInt(installHash.slice(0, 8), 16);
  const expectedPrefix = hashNum.toString(36).toUpperCase().padStart(8, '0').slice(-8);
  
  if (keyPrefix !== expectedPrefix) return { ok: false, error: 'Key does not match this installation.' };

  const durationDays = parseInt(durationStr, 10); // 000 = lifetime
  const activation = btoa(JSON.stringify({ activatedAt: Date.now(), durationDays }));
  localStorage.setItem(LICENSE_KEY, activation);
  return { ok: true };
}

// Helper for admin: given a hash, produce the activation key

How the admin generates the key: The admin receives the install hash via email. To produce the activation code:

Take first 8 hex chars of the hash → convert to integer → convert to Base36 → uppercase → take last 8 chars → pad to 8
Append - and 3-digit duration (030, 365, 000 for lifetime)
Result looks like: K7X2M9PQ-030
This is documented in a ADMIN_LICENSE_GUIDE.md (committed to the repo root, visible to the admin when they pull).

Verification
Fresh install: Clear localStorage entirely → login → banner appears with 30 days remaining; check that imharsh080@gmail.com received the install notification email.
Admin bypass: Login as imharsh080@gmail.com → no banner, full access.
Expiry simulation: Set enqboss_install_ts = btoa(String((Date.now() - 31*86400000) ^ 0xDEADBEEF)) in DevTools → refresh → full-screen expiry wall appears.
License activation: Compute the expected key from the hash in DevTools (see formula in license.ts), enter it → wall disappears, app loads.
Wrong key: Enter a random key → "Key does not match this installation" error shown.
Non-company email: Login with someone@gmail.com (not admin) → signed out with access-restricted error.
Duration suffix 000: Enter a XXXXXXXX-000 key → app stays licensed indefinitely (no expiry check).
